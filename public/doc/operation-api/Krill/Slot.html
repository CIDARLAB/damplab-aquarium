Overview

Instances of Slot represent elements of a [Collection]. The most likley way you will encounter a slot is when iterating over a collection or [CollectionArray].

Each slot has a row, column, and sample id, and you can test whether it is empty or nonempty.

Examples:

Show the contents of a collection, slot by slot

show do 
  collection_array.slots do |slot|
    if slow.nonempty?
      note "#{slot.collection.id}(#{slot.row},#{slot.col}) = #{slot.sample_id}"
     end
   end
 end
