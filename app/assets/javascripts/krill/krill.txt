KrillStep : A step, which contains many Parts

  - parts : a list of parts
  - disable : calls disable on all parts
  - history_item : returns whatever needs to be displayed to show the item in the history list

KrillPart : Part of a step, as in the title, a note, or a select

  - element : returns the jquery element for the part
  - disable : disables the part
  - get : gets the users input
